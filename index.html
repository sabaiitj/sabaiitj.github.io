Sarcasm Detection

For my Data Science Immersive Capstone, I analyzed over 26,000 news headlines to create a classification model for detecting sarcasm. I got the best accuracy of 80% with a BERT model. Classical machine learning algorithms did not perform very well on this data with accuracy scores in 62-65% range. Sarcasm detection is a very specialized study subject in NLP, a type of sentiment analysis where the focus is on recognizing sarcasm rather than detecting a sentiment throughout the entire spectrum. The first issue we have is that, unlike sentiment analysis, where sentiment categories are clearly defined , the limits of sarcasm are not so well defined. The applications of this task include designing intelligent chatbots, adding emotions into voice detection systems as also finding sarcastic reviews written by angry users on e-commerce websites.

Understanding what makes a Popular Reddit Post

For this project, I wanted to utilize natural language processing and various machine learning models to predict whether a reddit post will be popular or not. To test this, I used the PRAW API to pull over 10,000 posts from reddit. I was able to achieve an accuracy of 79% with a bagging classifier. I found that it's better to post in a subreddit with high subscriber count and tag as many high value subreddits as possible. Politics, games, funny,mademesmile,antiwork etc are popular topics to be tagged.
